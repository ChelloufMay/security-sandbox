# Code of Conduct

## Our Pledge
We want this project to be welcoming and friendly to everyone. We pledge to make participation a harassment-free experience for everyone.

## Expected Behavior
- Be respectful and considerate in language and actions.
- Assume good faith; offer constructive feedback.
- Follow repository contribution guidelines (issues, PRs).

## Unacceptable Behavior
- Harassment, intimidation, hateful language, or personal attacks.
- Deliberate disruption of discussions or project work.
- Posting private/personal information.

## Reporting
If you experience or witness unacceptable behavior, please contact the repository owner:
- Email: maychellouf28@gmail.com
- Or open an issue and add the label `private` (if public issues are unsuitable).

## Enforcement
Project maintainers may remove content or ban contributors who violate this code of conduct.
